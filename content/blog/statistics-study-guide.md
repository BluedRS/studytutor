## 📚 第一部分：核心概念梳理

### 1. 描述统计

**必须掌握的概念：**

| 概念 | 含义 | 应用场景 |
|-----|------|---------|
| 均值（Mean） | 数据的平均水平 | 描述集中趋势 |
| 中位数（Median） | 中间值，不受极端值影响 | 数据有异常值时 |
| 标准差（SD） | 数据离散程度 | 描述波动大小 |
| 方差（Variance） | 标准差的平方 | 统计推断基础 |

**R 语言代码：**
```r
# 计算描述统计
mean(data$variable)      # 均值
median(data$variable)    # 中位数
sd(data$variable)        # 标准差
var(data$variable)       # 方差
summary(data$variable)   # 综合描述
```

### 2. 概率分布

**常见分布：**

- **正态分布**：最常见，中心极限定理基础
- **t 分布**：小样本推断
- **卡方分布**：拟合优度检验
- **F 分布**：方差分析

### 3. 假设检验

**核心步骤：**

1. 设定原假设 H₀ 和备择假设 H₁
2. 选择显著性水平α（通常 0.05）
3. 计算检验统计量
4. 计算 p 值
5. 做出决策（p<α 拒绝 H₀）

**常见误解：**
- ❌ p 值是原假设为真的概率
- ✅ p 值是在 H₀ 为真时，观察到当前数据的概率

### 4. 回归分析

**简单线性回归：**
```
Y = β₀ + β₁X + ε
```

**关键输出解读：**
- **系数（Coefficient）**：X 每变化 1 单位，Y 变化多少
- **p 值**：系数是否显著不为 0
- **R²**：模型解释了多少变异
- **F 统计量**：整体模型是否显著

## 📝 第三部分：作业技巧

### 数据分析作业模板

**标准结构：**

1. **引言**（研究问题、数据来源）
2. **描述统计**（表格 + 图表）
3. **方法**（使用的统计方法）
4. **结果**（输出表格 + 解读）
5. **结论**（回答研究问题）
6. **附录**（代码 + 额外分析）

### 常见错误及避免

| 错误 | 后果 | 避免方法 |
|-----|------|---------|
| 不检查假设条件 | 结论无效 | 先做正态性、方差齐性检验 |
| p 值解读错误 | 被扣分 | 记住 p 值的正确定义 |
| 忽略效应量 | 分析不完整 | 报告 Cohen's d、η²等 |
| 图表不规范 | 印象分低 | 遵循学术图表标准 |

## 🎓 第四部分：考试备考

### 考前 2 周复习计划

**第 1-3 天：** 概念复习
- 过一遍所有核心概念
- 制作概念卡片

**第 4-7 天：** 做题练习
- 历年真题
- 课后习题

**第 8-10 天：** 模拟测试
- 限时完成一套试卷
- 分析错题

**第 11-14 天：** 查漏补缺
- 重点复习薄弱环节
- 休息调整

### 考试技巧

1. **先做简单题**：建立信心，保证基础分
2. **注意时间分配**：不要在一道题上花太多时间
3. **写出计算过程**：即使结果错了也有步骤分
4. **检查单位**：确保答案单位正确
