# 🚀 网站搭建指南

## 从零到上线的完整步骤

**预计时间：** 2-3 小时
**难度：** 中等（需要基本命令行操作）
**成本：** ¥0（GitHub Pages 免费）

## 📋 前置准备

### 1. 注册 GitHub 账号

访问 https://github.com 注册账号

**记住你的 GitHub 用户名**，网站 URL 将是：
```
https://你的用户名.github.io/studytutor/
```

### 2. 安装 Git

**Mac：**
```bash
# 安装 Homebrew（如果没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Git
brew install git
```

**Windows：**
下载 https://git-scm.com/download/win

### 3. 安装 Hugo

**Mac：**
```bash
brew install hugo
```

**Windows：**
```bash
# 使用 Chocolatey
choco install hugo -confirm

# 或手动下载 https://github.com/gohugoio/hugo/releases
```

### 4. 配置 Git

```bash
git config --global user.name "你的 GitHub 用户名"
git config --global user.email "你的邮箱"
```

## 🏗️ 搭建步骤

### 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名：`studytutor`
3. 可见性：**Public**（GitHub Pages 要求）
4. **不要**勾选"Add a README file"
5. 点击"Create repository"

### 步骤 2：初始化本地仓库

```bash
# 进入网站目录
cd /Users/ainiuma/Documents/openclaw/state/workspace/留学生商科辅导网站

# 初始化 Git
git init

# 添加所有文件
git add -A

# 提交
git commit -m "initial commit"

# 添加远程仓库（替换为你的用户名）
git remote add origin https://github.com/你的用户名/studytutor.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 步骤 3：配置 GitHub Pages

1. 访问你的仓库：https://github.com/你的用户名/studytutor
2. 点击 **Settings** → **Pages**
3. Source 选择 **GitHub Actions**
4. 保存

### 步骤 4：等待部署

GitHub Actions 会自动构建和部署网站，约 2-5 分钟。

**查看部署状态：**
1. 访问仓库的 **Actions** 标签
2. 点击正在运行的 workflow
3. 等待全部变绿 ✓

**访问网站：**
```
https://你的用户名.github.io/studytutor/
```

### 添加微信二维码

1. 生成微信二维码图片（可用草料二维码等工具）
2. 保存为 `wechat-qr.png`
3. 放到 `static/images/` 目录
4. 提交推送：
```bash
git add -A
git commit -m "add wechat qr code"
git push
```

### 自定义域名（可选）

**购买域名：**
- Namecheap、GoDaddy 等
- 建议：`.com` 或 `.io` 域名，约¥60/年

**配置 DNS：**
1. 添加 CNAME 记录：
   ```
   类型：CNAME
   主机：@ 或 www
   值：你的用户名.github.io
   ```

**配置 Hugo：**
编辑 `hugo.toml`：
```toml
baseURL = "https://你的域名.com/"
```

**配置 GitHub：**
1. 在仓库创建 `static/CNAME` 文件
2. 内容：`你的域名.com`
3. 提交推送

## ⚠️ 常见问题

### 问题 1：推送失败

**错误：** `remote: Repository not found`

**解决：** 检查远程仓库 URL 是否正确
```bash
git remote -v
git remote set-url origin https://github.com/你的用户名/studytutor.git
```

### 问题 2：网站 404

**原因：** GitHub Pages 还未部署完成

**解决：** 等待 Actions 完成，或检查 Pages 设置

### 问题 3：主题不显示

**原因：** Hugo 主题 submodule 未正确初始化

**解决：**
```bash
git submodule update --init --recursive
git push
```

### 问题 4：图片不显示

**检查：**
1. 图片是否在 `static/images/` 目录
2. Markdown 中路径是否正确：`![描述](/images/图片名.png)`
